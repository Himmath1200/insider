About Data Exfiltration

Data Exfiltration is the unauthorized transfer of data outside an organization.

Risk Factors:
  • Large file downloads (> 5MB)
  • Transfers to external destinations (USB, Email, Cloud)
  • Unusual access patterns or timing
  • Sensitive file access
Prevention Methods:
  • Data Loss Prevention (DLP) policies
  • Monitor file transfers
  • Restrict USB access
  • Encryption and access controls
